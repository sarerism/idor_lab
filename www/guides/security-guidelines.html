Password Requirements

  • Minimum 12 characters in length
  • Must include uppercase and lowercase letters
  • Must contain at least one number
  • Must include at least one special character
  • Cannot reuse last 5 passwords
  • Must be changed every 90 days
‚úÖ Best Practices:
  • Use a password manager to generate and store complex passwords
  • Never share your password with anyone, including IT staff
  • Use different passwords for different systems
  • Enable multi-factor authentication wherever available

üìß Email Security

Phishing Awareness

Be vigilant against phishing attempts. Warning signs include:

  • Urgent requests for personal information
  • Suspicious sender addresses
  • Unexpected attachments or links
  • Poor grammar or spelling
  • Requests to bypass normal procedures

Safe Email Practices

  • Verify sender identity before clicking links
  • Hover over links to preview destinations
  • Don't open unexpected attachments
  • Report suspicious emails to security team
  • Use encryption for sensitive information

üíª Workstation Security

  • Screen Lock: Always lock your computer when leaving your desk (Windows+L or Ctrl+Alt+L)
  • Auto-Lock: Screens automatically lock after 5 minutes of inactivity
  • Clean Desk: Don't leave sensitive documents visible
  • Software Updates: Install security patches promptly
  • Antivirus: Keep security software active and updated

üì± Mobile Device Security

  • Enable device encryption
  • Use strong PIN or biometric authentication
  • Install only approved applications
  • Report lost or stolen devices immediately
  • Don't store sensitive data on personal devices

üåê Network Security

Approved Networks

  • Company wireless network (secured with WPA3)
  • Wired office network connections
  • Authorized VPN for remote access
üö´ PROHIBITED:
  • Using public Wi-Fi for work activities
  • Connecting unauthorized devices to company network
  • Sharing network credentials
  • Bypassing security controls

üìÇ Data Protection

Data Classification

  • Public: Information approved for public release
  • Internal: General business information for employee use
  • Confidential: Sensitive business information with limited access
  • Restricted: Highly sensitive data requiring special handling

Data Handling

  • Store sensitive data only on approved systems
  • Encrypt confidential information in transit and at rest
  • Use secure file sharing services
  • Don't email sensitive data without encryption
  • Dispose of data securely (shred physical documents)

  • Phishing attempts
  • Unauthorized access attempts
  • Data breaches or leaks
