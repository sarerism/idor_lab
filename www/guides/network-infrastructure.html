Directory Services

LDAP-based authentication system provides centralized user management and single sign-on capabilities across all MBTI platforms.

  • Protocol: LDAP v3
  • Domain: mbti.local
  • Service Port: 389 (LDAP), 636 (LDAPS)

File Sharing

Network File System (NFS) provides shared storage for documentation, reports, and collaborative workspaces.

  • Protocol: NFSv4
  • Mount Points: Various departmental shares
  • Access: Authenticated users only

Web Services

Apache-based web servers host the employee portal and internal applications.

  • Primary Portal: http://portal.mbti.local
  • Load Balancing: Round-robin DNS
  • SSL/TLS: Enforced for external access
